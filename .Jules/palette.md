## 2026-05-24 - [Visual Feedback for CLI Tools]
**Learning:** CLI scripts often lack visual cues for data density. Adding ASCII progress bars and color-coded status provides immediate, intuitive understanding of resource usage that raw numbers alone don't convey. This improves the "glanceability" of the tool.
**Action:** Use ASCII bars and semantic coloring (Green/Yellow/Red) when displaying percentages or resource levels in CLI tools.
